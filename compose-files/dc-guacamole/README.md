# Apache Guacamole

Apache Guacamole is a clientless remote desktop gateway.

Official docs: [https://guacamole.apache.org/doc/gug/](https://guacamole.apache.org/doc/gug/).
Image docs: [https://github.com/jwetzell/docker-guacamole](https://github.com/jwetzell/docker-guacamole)

Default username and password: `guacadmin` / `guacadmin`
